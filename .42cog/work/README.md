# 工作记录 (Work)

> 记录项目开发过程中的关键决策、问题解决和迭代历程

## 重要说明

> **代码生成目录**: 实际生成的代码文件通常放在 `../src/` 目录下
>
> 项目结构示例：
> ```
> project-root/
> ├── .42cog/          # 认知敏捷法核心文件
> │   ├── real/        # 现实约束
> │   ├── cog/         # 认知模型
> │   ├── spec/        # 规约文档
> │   └── work/        # 工作记录（本文件）
> ├── .42plugin/       # 技能插件
> └── src/             # 实际代码目录 ⭐
>     ├── app/         # Next.js 应用目录
>     ├── components/  # React 组件
>     ├── lib/         # 工具库
>     └── ...          # 其他代码文件
> ```

## 工作日志格式
