# 项目元信息 (Meta)

**负责人**: xiaoyi

## 项目描述

LearnHub 是一个虚拟商品自营商城，专注于在线课程、会员订阅和数字资源的销售。平台同时服务 C 端个人用户和 B 端企业客户，为企业提供员工培训和团体订阅解决方案。

## 目标用户

- C 端个人用户: 希望学习新技能、获取数字资源的个人
- B 端企业用户: 需要为员工采购培训课程、管理学习进度的企业

## 核心价值主张

1. 优质内容：精选高质量在线课程和数字资源
2. 灵活订阅：支持单品购买和会员订阅多种模式
3. 企业友好：提供企业培训管理、团体订阅、学习数据分析

## 技术栈

**前端**:
- 框架: Next.js 15
- UI: Tailwind CSS + shadcn/ui
- 语言: TypeScript

**后端**:
- 运行时: Bun / Node.js
- 数据库: PostgreSQL
- ORM: Drizzle ORM

**基础设施**:
- 托管: Vercel / EdgeOne Pages
- 数据库: Neon / Supabase
- 认证: Better Auth

**支付**:
- 微信支付
- 支付宝

**AI集成**:
- AI SDK: Vercel AI SDK
- 模型: Claude / GPT-4
